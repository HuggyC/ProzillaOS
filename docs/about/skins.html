Skins ​

The look and feel of ProzillaOS can be customized using skins. Skins mainly consist of custom assets and stylesheets.

For more information about how to use skins, read the documentation. You can find the Figma file for the assets of ProzillaOS here.

Default ​

This is the default skin for ProzillaOS. It is inspired by Windows and Ubuntu Linux.
